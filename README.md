# GPU Video Compressor

動画ファイルのファイルサイズを劇的に軽量化するための、Windows右クリックメニュー統合型の圧縮ツールです。

NVIDIA製GPUのハードウェアエンコード（NVENC）を最大限に活用し、画質の劣化を最小限に抑えながら超高速で動画を圧縮します。Discordや各種SNSへのアップロード時、動画の保管容量を節約したい時など、素早く簡単に動画を軽量化したい場面に最適なソフトウェアです。

## 主な機能

- **超高速 GPU エンコード**
  NVIDIA NVENC (H.264, HEVC, AV1対応) を利用し、CPUに負荷をかけずに短時間で動画を軽量化します。デコード（読み込み）にもGPUを使用するため、非常に高速です。
- **目標ファイルサイズ指定 (容量優先モード)**
  「50MB以内に収める」「10MBにする」など、目標とするファイルサイズを指定するだけで、ツールが自動的に最適なビットレートを計算し、指定容量に収まるように圧縮します。
- **右クリックメニューからの即時実行 (プリセット連携)**
  動画ファイルを右クリックして表示されるプリセット（「Discord Nitro Basic用」「X用」など）を選ぶだけで、即座に最適な設定で軽量化処理が開始されます。
- **直感的な GUI とプリセット管理**
  解像度、フレームレート、画質（CQP）などの細かい設定を専用のGUIから直感的に変更できます。また、よく使う設定を保存したり、アプリ内からプリセットの名前を変更・削除できる管理機能（GUI）も備えています。

## 動作要件

- **OS**: Windows 10 / 11
- **GPU**: NVIDIA 製グラフィックボード (GTX 1660 以降など、NVENC 対応のハードウェア)
- **環境**: Python 3.x
- **依存ソフトウェア**: FFmpeg (環境変数 `PATH` に登録されている、またはシステムで利用可能であること)

## セットアップ手順

### 1. 右クリックメニューへの登録
本ツールを右クリックメニューに登録します。管理者権限は不要です。

1. コマンドプロンプトや PowerShell を開きます。
2. 本ツールのフォルダに移動し、以下のコマンドを実行します：
   ```bash
   python register_menu.py --register
   ```
   ※ 単純に `python register_menu.py` を実行すると、登録・解除を行うための専用画面が起動します。

### 2. アンインストール（メニューの解除）
右クリックメニューから本ツールを削除したい場合は、以下のコマンドを実行します：
```bash
python register_menu.py --unregister
```

## 使い方

### 右クリックから素早く圧縮（プリセット使用）
1. 圧縮したい動画ファイル（`.mp4`, `.mkv`, `.mov` など）を右クリックします。
2. Windows 11 の場合は「その他のオプションを表示」をクリックし、**「GPU動画コンバーター」** の中から「Discord用」などの目的に合ったプリセットを選択します。
3. 自動的に設定が読み込まれ、超高速で圧縮が開始されます。

### 詳細設定からの圧縮・プリセット管理
1. 右クリックメニューの「GPU動画コンバーター」から **「設定...」** を選択するか、プログラムを直接起動すると、GUI画面が表示されます。
2. 画面内で、出力コーデック、フレームレート、解像度、画質（品質優先 / 容量優先）などを細かく調整できます。
3. **「⚡ 圧縮開始」** ボタンをクリックすると変換が開始されます。
4. **「🎁 プリセット作成」** ボタンを押すと、現在の設定を新しいプリセットとして保存し、次回から右クリックで使えるようになります。
5. **「📋 プリセット管理」** ボタンからは、登録されているすべてのプリセットの名前変更や削除が行えます。

変換後のファイルは、元のファイルと同じフォルダに連番付きで保存されます。
